My personal browser homepage [hsiii.github.io/homepage](https://hsiii.github.io/homepage) built with [Vite](https://vite.dev/) and [React](https://react.dev/).

## Installation (requires [Node.js](https://nodejs.org/) 20.x)

1. Clone the repository:
    ```bash
    git clone https://github.com/Hsiii/homepage.git
    ```
2. Navigate to the project directory:
    ```bash
    cd homepage
    ```
3. Install dependencies:
    ```bash
    npm install
    ```
4. Run the development server:
    ```bash
    npm run dev
    ```

## Building for production

1. Generate a production build:
    ```bash
    npm run build
    ```
2. Preview the site:
    ```bash
    npm run preview
    ```
3. Deploy to GitHub Pages (deploy to `release` branch):
    ```bash
    npm run deploy
    ```
